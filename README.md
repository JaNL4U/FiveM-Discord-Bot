<h2>FiveM Discord Bot</h2>

<h4>Description</h4>
<p>This discord bot adds 2 commands to your discord server. The first command is a simple <code>!checkplayer FIRSTNAME LASTNAME</code> command where you get the money and phone number as result back.<br>
The second command is a <code>!checkplate PLATE</code> command where you get the owner back and aswell the phone number, sex and name.</p>
<h4>Installation</h4>
<p>Download the repository or clone it into your existing folder. After that setup the bot. You can do that in the <code>config.json</code> and <code>trans.json</code>. Open your command line tool for example cmd on windows. Navigate to the folder where the bot is located. Type into the console <code>node .</code> The bot should start and work fine. Try out your new bot :)</p>
